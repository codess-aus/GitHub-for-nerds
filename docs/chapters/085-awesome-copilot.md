# 08.5 Awesome Copilot

![Awesome Copilot slide](../assets/8.5-awesome.png)

## What it is

Awesome Copilot is a community-curated collection of reusable prompts, instructions, and plugins.

## Install and access

- Gallery: [awesome-copilot.github.com](https://awesome-copilot.github.com)
- Repository: [github.com/github/awesome-copilot](https://github.com/github/awesome-copilot)
- CLI install example: `copilot plugin install <name> @awesome-copilot`

## Exercise

Find one prompt or plugin and adapt it to your team's repo workflow.
