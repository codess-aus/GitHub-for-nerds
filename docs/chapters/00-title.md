# 00. GitHub for Microsoft Nerds

![Title slide: GitHub for Microsoft Nerds](../assets/0-title.png)

## What this chapter is about

This workshop is a practical tour of modern GitHub workflows for teams:

- core concepts: Git, repositories, pull requests
- AI assistance in the editor and terminal
- spec-driven and SDK-driven ways to build faster
- day-to-day tricks you can use immediately

## Attendee outcome

By the end, attendees should be able to move from idea to shipped change using GitHub-native tooling with confidence.
