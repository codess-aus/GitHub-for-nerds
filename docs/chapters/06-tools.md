# 06. Part 2 - Install and Where to Get Access

![Part 2 tools slide](../assets/6-tools.png)

## In this section

You will install or verify access for:

- GitHub Copilot
- Copilot CLI
- Awesome Copilot extensions
- Git-Ape
- GitHub Spec-Kit
- GitHub Copilot SDK
- GitHub Copilot App
- GitHub Mobile

## Quick setup checklist

1. Confirm your GitHub account and Copilot plan.
1. Install tooling in the order above.
1. Run one command or task per tool to confirm it works.
