## Why this matters first

Before AI tools, your team needs shared language for change management.

- Git is your local history engine
- repositories are the project home
- pull requests are the collaboration and quality gate

## Exercise

In pairs, explain these in one sentence each:

- Git
- repository
- pull request

If you can explain them clearly, the rest of the workshop becomes much easier.
