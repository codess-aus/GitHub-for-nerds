# 13. GitHub Mobile

![GitHub Mobile slide](../assets/13-githubmobile.png)

## What it is

GitHub Mobile brings pull request review, issue triage, and Copilot chat to your phone.

## Install and access

- iOS: App Store
- Android: Google Play
- Website: [github.com/mobile](https://github.com/mobile)

## Exercise

Review one PR from your phone:

1. leave one comment
1. approve or request changes
1. follow up in desktop workflow
